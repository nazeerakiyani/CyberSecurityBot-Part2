### Description
WPF chatbot that educates South African citizens on cybersecurity topics including passwords, phishing, privacy, scams, malware, and safe browsing. The application features a graphical user interface with voice greeting, ASCII art, dynamic responses, conversation flow, memory and recall, sentiment detection, and comprehensive error handling.

---

### Features
- **Voice greeting** on startup
- **ASCII art** logo display
- **Keyword recognition** (6 topics, 30+ variations)
- **Sequential random responses** with no-repeat logic
- **Conversation flow** with proactive suggestions after 3 tips
- **Memory and recall** (name + favourite topic personalisation)
- **Sentiment detection** (worried, curious, frustrated, neutral)
- **Error handling** for empty/unknown inputs
- **Input validation** to prevent crashes

---

### How to Run
1. Open `CyberSecurityBot.sln` in Visual Studio 2022
2. Press **F5** to build and run
3. Type your name when prompted
4. Ask about cybersecurity topics or say "tell me more"
5. Type "bye" or close the window to exit
